##  Project Overview
This project is a responsive multi-page website developed for Durban Tech Repairs.  
It forms part of the WEDE5020 Portfolio of Evidence (POE) and demonstrates the use of HTML, CSS, JavaScript, and SEO principles.

The website provides information about tech repair services, including an enquiry system and contact functionality.



##  Website Features

- Homepage with business introduction
- About page with company information
- Services page with repair offerings
- Enquiry form for customer requests
- Contact page with validation and business details
- Interactive navigation menu (hamburger menu)
- JavaScript form validation
- SEO optimisation (meta tags + structure)
